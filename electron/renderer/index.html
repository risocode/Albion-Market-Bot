Quick Help

  • Start scraping: Choose category → Start Category Fetch → pick city → confirm

  • View results: Open Fetched Prices tab for Live + CSV history

  • Ctrl+Shift+S Stop Category Fetch

  • Ctrl+Shift+2 Open Fetched Prices tab
